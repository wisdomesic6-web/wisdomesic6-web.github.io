01 AI Automation · Real Estate

Solwise Realty AI Agent

End-to-end real estate lead generation & qualification system — Lagos, Nigeria

Problem

Agents were manually sourcing leads from Google Maps, vetting them by hand, and following up inconsistently — hours of daily effort with no scoring system or structured CRM.

Solution

Built a fully automated pipeline: Apify scrapes property leads → Claude API scores each lead on budget, intent, timeline, and location → hot leads trigger automated WhatsApp follow-up sequences. A Vapi.ai voice agent and Botpress chatbot handle all inbound inquiries.

Result

Replaced 5 manual tools with one connected workflow. Agents now spend time exclusively on pre-qualified, high-intent leads — saving hours of daily prospecting.

03 AI Automation · Finance & Compliance

Automated Audit & Compliance System

AI-powered financial audit workflow — anomaly detection, report generation & stakeholder delivery

Problem

Finance teams were spending days pulling records, running consistency checks, and writing audit summaries manually — a process that was slow, error-prone, and hard to scale.
